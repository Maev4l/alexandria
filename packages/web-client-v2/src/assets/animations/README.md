# Onboarding Animations

Download the following Lottie animations from [LottieFiles](https://lottiefiles.com) and place them in this folder.

## Required Files

| File | Screen | Recommended Search |
|------|--------|-------------------|
| `welcome.json` | Welcome screen | "reading book", "library", "bookshelf" |

## How to Download

1. Go to [LottieFiles Free Animations](https://lottiefiles.com/free-animations)
2. Search for the recommended term

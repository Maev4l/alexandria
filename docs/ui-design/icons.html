Icon set — “The Shelf” proposals

A small circulation-and-paper icon family, narrating a real library’s life cycle.

The personality comes from a few bookish accents — book / book-plus / book-up · book-down (lend/return) / stamp (the date-due history card) — plus a “writing” pair (pen-line manual entry, pencil-line edit). Everything universal (chevrons, magnifier, trash, eye, spinner) stays conventional so the set reads instantly at 20px. Icons below are rendered live from lucide 0.469 (the app’s exact version). (“books” was the ideal collection glyph but isn’t in 0.469, so it’s replaced by layers/library.)
